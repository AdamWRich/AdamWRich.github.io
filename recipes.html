RecipeShare

sharing/blog site

One of my more complex builds, this was an assignment focues on full-fledged Flask capability with connection to a MySQL database. Users can register and log in (with form validations) as well as create (post) recipes and view ones made by others. Users can also edit and delete their own recipes.

This was a really fun project, and got me more interested in how backend code can change front end templates! Check out the code for this project on GitHub!
